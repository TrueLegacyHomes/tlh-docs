# Daily Workflow

How to use Claude Code with TLH skills in your everyday work.

## The 30-Second Version

1. **Open Claude Code**
2. **Pick a skill** from the skill picker
3. **Ask Claude** what you need
4. **Get work done**

That's it. The skill gives Claude the context it needs to help you.

---

## Step-by-Step

### 1. Open Claude Code

Launch Claude Code from your applications or terminal.

[SCREENSHOT: Claude Code app icon]

### 2. Pick a Skill

Click the skill picker (top of the chat) and choose the skill that matches your task:

| Task | Skill to Use |
|------|--------------|
| Find an answer about TLH | `solution-finder` |
| Work with data or reports | `data` |
| Draft customer responses | `customer-support` |
| Check brand guidelines | `brand-design-system` |
| Build a dashboard | `reporting-dashboards` |
| Create a new process doc | `process-playbook-builder` |

[SCREENSHOT: Skill picker dropdown]

### 3. Chat Naturally

Just ask Claude what you need in plain English:

> "What's our policy on estate sale minimums?"

> "Help me draft a response to this customer complaint"

> "Create a report showing sales by region for Q1"

Claude will use the skill's knowledge to help you.

### 4. Follow Up

Keep the conversation going:

> "Make it more friendly"

> "Add a column for year-over-year comparison"

> "What else should I include?"

## Common Workflows

### Finding Answers
1. Select `solution-finder`
2. Ask your question
3. Claude searches TLH knowledge and responds

### Drafting Customer Emails
1. Select `customer-support`
2. Paste the customer's message
3. Ask Claude to draft a response
4. Review and send

### Creating Reports
1. Select `data` or `reporting-dashboards`
2. Describe what you need
3. Claude builds it
4. Export or share

### Checking Brand Guidelines
1. Select `brand-design-system`
2. Ask about colors, fonts, tone, etc.
3. Get the official TLH answer

---

## Tips for Better Results

### Be Specific
❌ "Help me with this"  
✅ "Help me write a follow-up email to a family who attended our estate sale last Saturday"

### Give Context
❌ "Write a report"  
✅ "Write a report showing our San Diego estate sales for the last 3 months, grouped by neighborhood"

### Ask for Changes
If the first response isn't right, just say:
- "Make it shorter"
- "Use a friendlier tone"
- "Add more detail about pricing"

---

## Switching Skills

You can switch skills anytime:

1. Click the skill picker
2. Choose a different skill
3. Continue working

Claude will now use the new skill's context.

::: tip
Start a new chat when switching to a completely different task. This keeps things clean.
:::
